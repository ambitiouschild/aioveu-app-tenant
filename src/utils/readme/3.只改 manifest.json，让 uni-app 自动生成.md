## 常见问题解答

### Q: 为什么 uni-app 不更新 project.config.json？

A: 可能原因：

1. **缓存问题**：删除 `node_modules/.cache`和 `unpackage`目录
2. **文件权限**：确保有写入权限
3. **HBuilderX 问题**：如果是用 HBuilderX，重启 IDE
4. **配置冲突**：检查是否有其他配置覆盖

### Q: 编译后 project.config.json 还是错的怎么办？

A: 强制重新生成：

```
# 清理所有编译产物
rm -rf unpackage node_modules/.cache
rm project.config.json

# 重新安装依赖（可选）
npm install

# 重新编译
npm run dev:mp-weixin
```

### Q: 我需要手动修改 project.config.json 的其他配置吗？

A: 不推荐。应该在 `manifest.json`的 `mp-weixin`节点下配置，或在 `vue.config.js`中配置：

```
// vue.config.js
module.exports = {
  transpileDependencies: ['uni-app'],
  pluginOptions: {
    'mp-weixin': {
      // 微信小程序相关配置
      appid: process.env.VUE_APP_MP_WEIXIN_APPID,
      setting: {
        urlCheck: false,
        es6: true
      }
    }
  }
};
```

- **不要手动修改生成产物**，否则会导致配置不一致
