## 一、用你现在的代码，逐层拆给你看（非常关键）

### ① `uni.request`返回的 `response`

```
uni.request({
  success: (response) => {
    // response 是 uni-app 的 HTTP 响应对象
  }
})
```

它的结构是：

```
response = {
  statusCode: 200,
  data: ...,        // 👈 后端返回的 JSON
  header: {},
  cookies: [],
  errMsg: "request:ok"
}
```

------

### ② `response.data`是什么？

```
response.data === 后端返回的 JSON
```

也就是：

```
{
  "code": 200,
  "msg": "success",
  "data": {
    "orderSn": "202606230000106449",
    "payParams": { ... }
  }
}
```

✅ 所以：

| 表达式                 | 含义             |
| ---------------------- | ---------------- |
| `response.data`        | 后端 Result 对象 |
| `response.data.code`   | 业务状态码       |
| `response.data.msg`    | 业务消息         |
| ✅ `response.data.data` | **业务数据本体** |

------

### ③ 你代码里是怎么用的？

```
const resData = response.data; // Result<T>
```

然后：

```
return Promise.resolve(resData.data); // ✅ 业务数据本体
```

👉 **你这一步就是在“解构出 `response.data.data`”**
